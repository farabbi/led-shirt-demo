# led-shirt-demo

## Instructions:

1. Clone this repo

```bash
git clone https://github.com/farabbi/led-shirt-demo.git
```

2. Get into the repo

```bash
cd led-shirt-demo
```

3. Build utils

```bash
bash build_utils.sh
```

4. Record a 1-minute video on Desktop

```bash
python3 record.py
```

5. Decode the video and play it on LED panel

```bash
bash decode-and-play.sh
```
